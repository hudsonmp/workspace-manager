Name Your Spaces

Click ⌂ → Rename This Space. The name shows in the menu bar and persists across switches.

API Reference

Backend runs at localhost:8001

POST /shelve Save tabs as a bundle
GET /bundles List all shelved bundles
POST /restore/{id} Restore a bundle
DEL /bundles/{id} Delete a bundle
GET /active-space Current Space name
PUT /spaces/{idx}/name Rename a Space
POST /suggest-name ML name suggestion

Architecture

Two processes, no Chrome extension. The menu bar app captures Cmd+Shift+K globally via NSEvent, reads Chrome tabs via AppleScript, and sends them to a local FastAPI backend. Naming uses TF-IDF keyword extraction + domain grouping. All data in local SQLite.
